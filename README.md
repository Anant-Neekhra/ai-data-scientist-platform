# AI Data Scientist Platform

End-to-end machine learning platform that automates EDA, preprocessing, model training, hyperparameter tuning, explainability (SHAP), experiment tracking (MLflow), and deployment (FastAPI + Streamlit) for tabular datasets.

🚧 **Under active development.** Full documentation coming soon.

## Tech Stack
Python · Pandas · Scikit-learn · XGBoost · LightGBM · CatBoost · TensorFlow · SHAP · MLflow · FastAPI · Streamlit · Docker
